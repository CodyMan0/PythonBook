# 문제 풀이

## 모험가 길드

### 문제의 아이디어 생각해낸 포인트

- 핵심은 정렬, 공포를 오름차순 정렬 , 순서대로 탐색
- 최소한의 인원

### 시간 복잡도 계산

A. O(n)

### 입력

```python
# 2회
## 핵심은 정렬


n = int(input())
array = list(map(int,input().split()))
array.sort()

result = 0
count = 0

print(array)
for horror in array :
    count += 1
    if count >= horror :
        result += 1
        count =0
    print(horror,result,count)

print(result)


```
